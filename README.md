# Line 財經爬蟲機器人

## 狀態
停止維護中，未上線。

## 功能
1.油價查詢。<br>
2.股價查詢。<br>
3.匯率查詢、轉換。<br>
4.記錄使用者關注的匯率，推送使用者關注的匯率資訊。<br>
